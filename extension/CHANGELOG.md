# Changelog

## [0.1.0] — 2025-07-14

### Added
- Initial release
- TextMate grammar for full Zink syntax highlighting
- Language configuration (brackets, comments, indentation, folding)
- 11 code snippets (fn, if, ife, while, loop, let, say, map, filter, reduce)
- `.zink` file association
- **Commands**: Run File, Run Selection, Open REPL, Restart REPL, Build Interpreter
- **Keybindings**: `Ctrl+Shift+R` (run), `Ctrl+Shift+Enter` (run selection), `Ctrl+Shift+Z` (REPL)
- **Context menus**: Editor title play button & right-click menu for `.zink` files
- **Settings**: `zink.interpreterPath`, `zink.runInIntegratedTerminal`, `zink.clearTerminalBeforeRun`
- **Zink logo** — SVG and PNG lightning-bolt icon
