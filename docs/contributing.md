# Contributing

See the [Contributing Guide](https://github.com/nicbarker/zink/blob/main/CONTRIBUTING.md) in the project root for detailed information on how to contribute to Zink.

## Quick Start

```bash
# Clone
git clone https://github.com/nicbarker/zink.git
cd zink

# Build the interpreter
cd lang && cargo build

# Run the playground locally
cd ../playground && npm install && npm run dev

# Build the docs locally
cd ../docs && bun install && bun run dev
```
